Search and tag page blocks can be used to display helpful feedback to visitors. The required markup is consistent, just remember to localize your messages.

Search pages

{block:SearchPage}
<div class="search-page message">
  {block:SearchResults}
  <p class="search-result">{lang:Search results for SearchQuery 2}</p>
  {/block:SearchResults}

  {block:NoSearchResults}
  <p class="search-result">{lang:Sorry no results for SearchQuery 2}</p>
  {/block:NoSearchResults}

  <span class="divider"></span>
</div>
{/block:SearchPage}

When there are search results, a localized success message is shown, conditionally rendered with {block:SearchResults}:

However, when there no results, {block:NoSearchResults} is used:

Tag pages

Tag pages are only created when a tag is present:

{block:TagPage}
<div class="tag-page message">
  <p class="tag-wrapper">{lang:Showing posts tagged Tag 2}</p>

  <span class="divider"></span>
</div>
{/block:TagPage}
